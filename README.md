# ExHailo

Run machine learning models on the Hailo AI accelerator.

This project is forked from https://github.com/vittoriabitton/nx_hailo and changed to work with [ExNVR](https://github.com/evercam/ex_nvr)

## Installation

The package can be installed by adding `ex_hailo` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:ex_hailo, "~> 0.1.0"}
  ]
end
```
